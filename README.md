## Expense Tracker
## Overview

This is a full-stack Expense Tracker application built using React (TypeScript) for the frontend and Python Flask for the backend.
The application allows users to add, view, filter, and delete expenses with real-time total calculations.

It demonstrates full-stack development skills including REST API design, database integration, and frontend state management.

## Tech Stack

Frontend

React with TypeScript

Axios for API calls

Basic CSS styling

Backend

Python Flask

SQLAlchemy ORM

SQLite database

Flask-CORS for frontend-backend communication

## Features

Add new expenses with description, amount, and category

Delete expenses

View all expenses in table format

Filter expenses by category

Filter expenses by date

Dynamic total calculation based on filters

RESTful API integration

Clean and responsive UI
